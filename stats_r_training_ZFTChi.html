Section 02

The t-test

t
Parametric · Continuous Data · Small Samples
Student's t-test

What Is It & Why Does It Exist?

Imagine you developed two wheat varieties — HD-2967 and PBW-550 — and measured grain yield (g/plant) in 15 plants each. You want to know: "Is the difference in their average yields real, or just random variation?"

The t-test was invented in 1908 by William Sealy Gosset (publishing as "Student") specifically for small sample sizes where the true population standard deviation (σ) is unknown — exactly your situation in a greenhouse experiment.

🧠 Memory Anchor

"Tiny samples, Two means, Test them" — the three T's of the t-test. When n < 30 and σ is unknown, reach for t.

The t Statistic — Intuition First

The t statistic measures how many standard errors apart two means are. The bigger the t value, the less likely the difference is due to chance.

Core Formula — One-Sample t
t = (x̄ - μ₀) / (s / √n)
x̄ = sample mean  |  μ₀ = hypothesised mean  |  s = sample std deviation  |  n = sample size
Degrees of freedom: df = n - 1
Two-Sample Independent t (Equal Variance)
t = (x̄₁ - x̄₂) / √[sp²(1/n₁ + 1/n₂)]
sp² = pooled variance = [(n₁-1)s₁² + (n₂-1)s₂²] / (n₁+n₂-2)  |  df = n₁+n₂-2
Paired t-test
t = d̄ / (s_d / √n)
d̄ = mean of differences  |  s_d = std deviation of differences  |  df = n - 1

Assumptions of the t-test

📐
Data are approximately normal (or n > 30)
🔗
Observations are independent
⚖️
Equal variances (for independent t, Welch's corrects this)
📏
Data is continuous (not counts or categories)

Three Flavours in Plant Breeding

TypeWhen to UseGenetics Example
One-sample Compare your sample mean to a known standard "Is our variety's protein content different from the national standard of 12%?"
Independent (two-sample) Compare means of two different groups "Does variety A produce more grain than variety B?"
Paired Same subject measured twice (before/after, same plant, two treatments) "Do fungicide-treated plants have higher yield than untreated plants measured in same plots?"

R Code — One-Sample t-test

Scenario: You grew 10 tomato plants with a new fertilizer. Average fruit weight was tested against the known industry standard of 200g.

R Language
one_sample_ttest.R
# ════════════════════════════════════════════════════
# ONE-SAMPLE t-TEST IN PLANT BREEDING
# Question: Is our new fertilizer changing tomato fruit
#           weight from the standard 200g?
# ════════════════════════════════════════════════════

# STEP 1: Enter your data
fruit_weight <- c(210, 198, 225, 215, 202,
                   218, 195, 230, 208, 220)

# STEP 2: Check normality — ALWAYS do this first!
shapiro.test(fruit_weight)
# If p > 0.05 → normal → proceed with t-test safely

# STEP 3: Explore data visually
par(mfrow = c(1, 2))  # 1 row, 2 columns of plots
hist(fruit_weight,
     main = "Fruit Weight Distribution",
     xlab = "Weight (g)",
     col  = "#3fb950",
     border = "white")
qqnorm(fruit_weight); qqline(fruit_weight, col = "red")

# STEP 4: Run the t-test
result <- t.test(
  fruit_weight,
  mu = 200,            # μ₀ = our null hypothesis value
  alternative = "two.sided",  # or "greater" / "less"
  conf.level = 0.95   # 95% confidence interval
)

# STEP 5: View full results
print(result)

# STEP 6: Extract specific values
cat("t statistic:", result$statistic, "\n")
cat("Degrees of freedom:", result$parameter, "\n")
cat("p-value:", result$p.value, "\n")
cat("95% CI: [", result$conf.int[1], ",",
    result$conf.int[2], "]\n")
cat("Sample mean:", result$estimate, "\n")

# STEP 7: Decision rule
if (result$p.value < 0.05) {
  cat("\nCONCLUSION: Significant! Mean differs from 200g.\n")
} else {
  cat("\nCONCLUSION: Not significant. No evidence against 200g.\n")
}

R Code — Independent Two-Sample t-test

Scenario: Compare grain yield (g/plant) of Variety A vs Variety B in a field trial with 12 plants each.

R Language
independent_ttest.R
# ════════════════════════════════════════════════════
# INDEPENDENT TWO-SAMPLE t-TEST
# Comparing grain yield of two wheat varieties
# ════════════════════════════════════════════════════

# STEP 1: Data entry
variety_A <- c(45.2, 48.1, 44.9, 47.3, 46.0, 49.2,
                45.8, 47.1, 48.5, 46.3, 44.7, 47.9)
variety_B <- c(42.1, 43.5, 41.8, 44.2, 42.9, 43.0,
                41.5, 44.8, 42.3, 43.7, 41.2, 44.1)

# STEP 2: Summary statistics
cat("Variety A — Mean:", mean(variety_A),
    "| SD:", sd(variety_A), "\n")
cat("Variety B — Mean:", mean(variety_B),
    "| SD:", sd(variety_B), "\n")

# STEP 3: Check normality for BOTH groups
shapiro.test(variety_A)
shapiro.test(variety_B)

# STEP 4: Check equality of variances (F-test)
# If p > 0.05 → variances are equal → use var.equal = TRUE
var.test(variety_A, variety_B)

# STEP 5: Run the t-test
# var.equal = TRUE → classical Student's t (pooled variance)
# var.equal = FALSE → Welch's t (safer when variances differ)
t_result <- t.test(variety_A, variety_B,
                    alternative = "two.sided",
                    var.equal = TRUE,
                    conf.level = 0.95)
print(t_result)

# STEP 6: Visualise (boxplot) — always plot your data!
all_yield <- c(variety_A, variety_B)
variety   <- rep(c("Variety A", "Variety B"),
                  each = 12)
boxplot(all_yield ~ variety,
        main = "Grain Yield: Variety A vs B",
        ylab = "Yield (g/plant)",
        col  = c("#3fb950", "#58a6ff"),
        border = "white",
        notch = TRUE)  # notches show 95% CI of median

# STEP 7: Effect size — Cohen's d (biological importance!)
# p-value tells significance, Cohen's d tells MAGNITUDE
pooled_sd <- sqrt((var(variety_A) + var(variety_B)) / 2)
cohen_d   <- (mean(variety_A) - mean(variety_B)) / pooled_sd
cat("Cohen's d =", round(cohen_d, 3),
    "(>0.8 = large effect)\n")

R Code — Paired t-test

Scenario: Same 10 rice plants — yield measured before and after drought-stress treatment.

R Language
paired_ttest.R
# ════════════════════════════════════════════════════
# PAIRED t-TEST
# Same 10 rice genotypes: yield before vs after
# drought stress application
# ════════════════════════════════════════════════════

before_stress <- c(32.1, 28.4, 35.2, 30.6, 33.8,
                    29.1, 31.5, 34.0, 28.9, 32.7)
after_stress  <- c(28.3, 24.1, 31.0, 26.4, 29.2,
                    25.5, 27.8, 30.1, 24.7, 28.9)

# The key: paired = TRUE makes R compute differences internally

                         paired = TRUE,
                         alternative = "two.sided")
print(paired_result)

# Show the differences
diff <- before_stress - after_stress
cat("\nYield reduction per plant:", diff, "\n")
cat("Average reduction:", mean(diff), "g/plant\n")

# Visualise the paired differences
plot(before_stress, after_stress,
     xlab = "Before Stress (g)",
     ylab = "After Stress (g)",
     main = "Paired Yield Data: Before vs After Drought",
     pch = 19, col = "#58a6ff")
abline(a = 0, b = 1, col = "red", lty = 2)
# Points below the diagonal line = yield dropped
💡
Pro Tip — p-value is NOT Enough! A p-value of 0.001 just means the effect is statistically significant — it doesn't tell you if it's biologically meaningful. Always report Cohen's d (effect size) and the 95% confidence interval. A 0.01g yield difference might be significant with 10,000 plants but completely useless to a plant breeder.
Section 03

The z-test

z
Parametric · Large Samples (n ≥ 30) · σ Known
z-test

What Is It?

The z-test is the elder sibling of the t-test. When your sample is large (n ≥ 30), by the Central Limit Theorem the sample means become normally distributed regardless of the raw data distribution. At this point, the t-distribution converges with the standard normal (z) distribution.

🧠 Memory Anchor

"Zero in on large populations — Z-test for n ≥ 30 or when σ is known from historical data." Think of z as t's confident, large-sample version.

In a national crop survey measuring leaf nitrogen content across 200 farms, or in a large seed quality study — z-test is appropriate.

z Statistic — One Sample
z = (x̄ - μ₀) / (σ / √n)
σ = known population standard deviation (or s when n is large)
Critical values: z = ±1.96 for α=0.05 (two-tailed) | z = ±2.576 for α=0.01

t-test vs z-test — When to Choose?

Criteriont-testz-test
Sample sizeAny (best for n < 30)n ≥ 30
σ (population std)Unknown — estimated from dataKnown from literature/history
Distribution usedt-distribution (df-dependent)Standard Normal (Z)
In Rt.test()BSDA::z.test() or manual

R Code — z-test (One Sample)

Scenario: 50 cotton plants from a new hybrid. Known population σ = 8g from 30 years of breeding data. Is our hybrid's seed weight different from 95g?

R Language
z_test.R
# ════════════════════════════════════════════════════
# z-TEST IN PLANT BREEDING
# Scenario: Cotton seed weight — new hybrid vs standard
# Population σ = 8g (known from 30 years of records)
# H₀: μ = 95g  |  H₁: μ ≠ 95g
# ════════════════════════════════════════════════════

# METHOD 1: Manual z-test (no package needed)
seed_weight <- c(
  97.2, 98.5, 96.1, 99.3, 97.8, 95.4, 100.1, 98.2,
  96.7, 99.0, 97.5, 98.8, 96.3, 97.1, 100.4, 98.9,
  97.6, 96.8, 99.2, 97.3, 98.1, 96.5, 97.9, 99.6,
  98.4, 97.0, 96.2, 99.8, 97.7, 98.6, 96.9, 97.4,
  98.3, 96.6, 99.1, 97.2, 98.7, 96.4, 99.5, 97.8,
  98.0, 96.1, 97.5, 99.3, 97.9, 98.5, 96.8, 97.2,
  99.0, 97.6
)

n       <- length(seed_weight)  # 50
x_bar   <- mean(seed_weight)
mu_0    <- 95                   # null hypothesis mean
sigma   <- 8                    # KNOWN population SD

# Compute z statistic
z_stat <- (x_bar - mu_0) / (sigma / sqrt(n))
cat("z statistic:", round(z_stat, 4), "\n")

# Two-tailed p-value using standard normal
p_value <- 2 * pnorm(-abs(z_stat))
cat("p-value:", round(p_value, 6), "\n")

# 95% Confidence interval
ci_lower <- x_bar - 1.96 * (sigma / sqrt(n))
ci_upper <- x_bar + 1.96 * (sigma / sqrt(n))
cat("95% CI: [", round(ci_lower, 2), ",",
    round(ci_upper, 2), "]\n")

# ─── METHOD 2: Using BSDA package (cleaner) ───
# install.packages("BSDA")  # run once
library(BSDA)
z.test(seed_weight,
       mu    = 95,
       sigma.x = 8,
       alternative = "two.sided",
       conf.level  = 0.95)

# Visualise: where does our z fall on the standard normal?
x_vals  <- seq(-4, 4, length.out = 500)
plot(x_vals, dnorm(x_vals),
     type = "l", lwd = 2, col = "white",
     xlab = "z", ylab = "Density",
     main = "z-test: Critical Region (α = 0.05)")
# Shade rejection regions
polygon(c(-4, seq(-4, -1.96, 0.01), -1.96),
        c(0, dnorm(seq(-4, -1.96, 0.01)), 0),
        col = "#f85149", border = NA)
polygon(c(1.96, seq(1.96, 4, 0.01), 4),
        c(0, dnorm(seq(1.96, 4, 0.01)), 0),
        col = "#f85149", border = NA)
abline(v = z_stat, col = "#3fb950", lwd = 2, lty = 2)
legend("topright",
       legend = c("Rejection region", paste("z =", round(z_stat, 2))),
       col    = c("#f85149", "#3fb950"),
       lty    = c(1, 2), lwd = 2)
Section 04

The χ² (Chi-Square) Test

χ²
Non-Parametric · Categorical Data · Counts/Frequencies
Chi-Square Test

The Geneticist's Best Friend

This test was born for genetics. When Gregor Mendel crossed pea plants and got 705 round : 224 wrinkled seeds, he needed a way to ask: "Is this 3:1 ratio real, or just chance?" That is exactly what χ² answers.

χ² tests work on counts/frequencies — not measurements. Categories like phenotype classes, genotype classes, resistance vs susceptibility — these are χ² territory.

🧠 Memory Anchor

"χ² = Counts, Categories, Chromosome ratios" — if you're counting plants into buckets (tall/short, resistant/susceptible, 3:1, 9:3:3:1), use chi-square.

Chi-Square Statistic
χ² = Σ [(O - E)² / E]
O = Observed counts (what you actually counted in the field)
E = Expected counts (what Mendel's ratios / independence predict)
Degrees of freedom = (rows - 1)(columns - 1) for contingency tables
df = k - 1 for goodness-of-fit tests (k = number of categories)

Two Types in Genetics

TypeQuestionGenetics Example
Goodness-of-Fit Do observed frequencies match expected theoretical ratios? "Do my F₂ plants follow the expected 9:3:3:1 dihybrid ratio?"
Test of Independence Are two categorical variables associated? "Is disease resistance independent of flower colour?"

Assumptions

🔢
Counts, not percentages or measurements
📊
Each expected cell count ≥ 5 (use Fisher's exact test if not)
🔗
Observations are independent
📋
Categories must be mutually exclusive

R Code — Goodness-of-Fit (Mendelian Ratios)

Scenario: You crossed two heterozygous wheat plants (Rr × Rr). From 400 F₂ progeny, you want to verify the expected 3:1 ratio (resistant:susceptible).

R Language
chi2_goodness_of_fit.R
# ════════════════════════════════════════════════════
# χ² GOODNESS-OF-FIT TEST — MENDELIAN SEGREGATION
# Cross: Rr × Rr (rust resistance gene)
# Expected F₂ ratio: 3 Resistant : 1 Susceptible
# Observed: 315 resistant, 85 susceptible (total=400)
# H₀: Observed frequencies fit a 3:1 Mendelian ratio
# ════════════════════════════════════════════════════

# STEP 1: Observed counts
observed <- c(315, 85)
names(observed) <- c("Resistant", "Susceptible")

# STEP 2: Expected probabilities from Mendelian law
expected_prob <- c(3/4, 1/4)   # 3:1 ratio

# STEP 3: Calculate expected counts manually
total    <- sum(observed)     # 400
expected <- total * expected_prob  # 300, 100
cat("Expected counts:", expected, "\n")
cat("Observed counts:", observed, "\n")

# STEP 4: Run chi-square goodness-of-fit test
chi_result <- chisq.test(observed,
                          p = expected_prob)
print(chi_result)

# STEP 5: Manual verification of chi-square value
chi_manual <- sum((observed - expected)^2 / expected)
cat("\nManual χ² =", round(chi_manual, 4), "\n")
# Should match chisq.test output

# STEP 6: Get residuals (which category deviated most?)
cat("\nPearson residuals:\n")
print(chi_result$residuals)
# Large residuals (|r| > 2) indicate problematic categories

# STEP 7: Visualise — Observed vs Expected
barplot(
  rbind(observed, expected),
  beside  = TRUE,
  names.arg = c("Resistant", "Susceptible"),
  col     = c("#3fb950", "#58a6ff"),
  legend  = c("Observed", "Expected"),
  main    = "F₂ Segregation: Observed vs Expected (3:1)",
  ylab    = "Number of Plants",
  border  = "white"
)

# ── DIHYBRID RATIO (9:3:3:1) — extend easily ──
dihybrid_observed <- c(248, 79, 78, 35)
names(dihybrid_observed) <-
  c("Round Yellow", "Round Green",
    "Wrinkled Yellow", "Wrinkled Green")
expected_9331 <- c(9, 3, 3, 1) / 16

chisq.test(dihybrid_observed, p = expected_9331)

R Code — Chi-Square Test of Independence

Scenario: Is disease resistance linked to leaf colour? 200 plants, classified in a 2×2 table.

R Language
chi2_independence.R
# ════════════════════════════════════════════════════
# χ² TEST OF INDEPENDENCE
# Are disease resistance and leaf colour associated?
# H₀: The two traits are independent (unlinked)
# ════════════════════════════════════════════════════

# Build the contingency table
plant_data <- matrix(
  c(80, 20,    # Resistant: green-leaf, yellow-leaf
    30, 70),   # Susceptible: green-leaf, yellow-leaf
  nrow = 2,
  dimnames = list(
    Disease    = c("Resistant", "Susceptible"),
    Leaf_Colour = c("Green", "Yellow")
  )
)
cat("Contingency Table:\n")
print(plant_data)

# Add margins to see row/column totals
addmargins(plant_data)

# Run chi-square test of independence
chi_indep <- chisq.test(plant_data)
print(chi_indep)

# Expected counts (must all be ≥ 5)
cat("\nExpected counts:\n")
print(chi_indep$expected)

# If any expected count < 5, use Fisher's Exact Test
fisher.test(plant_data)

# Cramér's V — effect size for chi-square
# install.packages("rcompanion") → run once
library(rcompanion)
cramerV(plant_data)
# 0 = no association, 1 = perfect association
# V > 0.3 = meaningful association in genetics

# Visualise as mosaic plot
mosaicplot(plant_data,
           main  = "Resistance × Leaf Colour Association",
           color = c("#3fb950", "#d29922"),
           shade = TRUE)  # shade = TRUE shows residual colours

Section 05

The F-test (Variance Ratio Test)

F
Parametric · Comparing Variances · Precursor to ANOVA
F-test / Variance Ratio Test

What Is It?

While the t-test compares means, the F-test compares variances. It asks: "Are the two groups equally variable, or is one genotype more inconsistent than the other?"

In plant breeding, stability matters. A variety that yields 50g ± 20g in all environments is less valuable than one yielding 50g ± 3g. Genotypic stability = low variance. The F-test quantifies this.

The F-test is also the backbone of ANOVA — ANOVA is literally an F-test applied to multiple groups.

🧠 Memory Anchor

"F = Fisher's ratio of variances — always larger variance on top. If F ≈ 1, the groups are equally variable. If F is very large, one group varies much more."

F Statistic
F = s₁² / s₂²
s₁² = variance of group 1 (by convention, the LARGER variance goes on top)
s₂² = variance of group 2
df₁ = n₁ - 1 (numerator df)  |  df₂ = n₂ - 1 (denominator df)
Under H₀ (variances equal), F follows an F-distribution

When to Use the F-test

✅
Before running an independent t-test (check equal variance assumption)
✅
Comparing genotypic stability across environments
✅
Quality control — is one seed batch more variable than another?
✅
As part of ANOVA and regression analysis

R Code — F-test for Equality of Variances

Scenario: Two soybean varieties. Which one is more stable in yield across 20 plots?

R Language
f_test_variance.R
# ════════════════════════════════════════════════════
# F-TEST: COMPARING YIELD VARIABILITY OF TWO SOYBEAN
# VARIETIES ACROSS 20 FIELD PLOTS
# H₀: σ₁² = σ₂²  (equal variances — one is not more
#                   stable than the other)
# H₁: σ₁² ≠ σ₂²
# ════════════════════════════════════════════════════

variety_JS335 <- c(
  28.2, 27.8, 29.1, 28.5, 27.9,
  28.3, 29.0, 28.1, 27.7, 28.9,
  28.4, 27.6, 29.2, 28.7, 28.0,
  28.6, 27.5, 29.3, 28.8, 28.2
)

variety_DS228 <- c(
  24.1, 31.5, 26.8, 33.2, 22.9,
  35.1, 25.3, 30.7, 23.4, 32.8,
  27.1, 29.5, 21.6, 34.0, 26.2,
  31.9, 24.8, 33.5, 28.4, 30.1
)

# STEP 1: Summary — variances side by side
cat("JS335 — Mean:", round(mean(variety_JS335), 2),
    "| Variance:", round(var(variety_JS335), 3), "\n")
cat("DS228 — Mean:", round(mean(variety_DS228), 2),
    "| Variance:", round(var(variety_DS228), 3), "\n")

# STEP 2: Run F-test
f_result <- var.test(variety_JS335, variety_DS228,
                      alternative = "two.sided",
                      conf.level  = 0.95)
print(f_result)

# STEP 3: Extract F statistic manually
F_stat <- var(variety_JS335) / var(variety_DS228)
cat("F statistic:", round(F_stat, 4), "\n")

# STEP 4: Levene's test (more robust — less sensitive
#         to non-normality than the F-test)
# install.packages("car")  # run once
library(car)
yield   <- c(variety_JS335, variety_DS228)
variety <- factor(rep(c("JS335", "DS228"), each = 20))
leveneTest(yield ~ variety)

# STEP 5: Visualise variance (side-by-side boxplot)
boxplot(yield ~ variety,
        main   = "Yield Stability: JS335 vs DS228",
        ylab   = "Yield (q/ha)",
        col    = c("#3fb950", "#d29922"),
        border = "white",
        notch  = TRUE)

# Coefficient of Variation (CV%) — breeder's stability index
CV_JS335 <- (sd(variety_JS335) / mean(variety_JS335)) * 100
CV_DS228 <- (sd(variety_DS228) / mean(variety_DS228)) * 100
cat("\nCV% JS335:", round(CV_JS335, 2), "%\n")
cat("CV% DS228:", round(CV_DS228, 2), "%\n")
# CV < 10% = stable variety (preferred by breeders)
ℹ️
F-test in Regression Analysis The F-test also appears in linear regression as the overall F-test, asking: "Does my regression model explain significantly more variance than just using the mean?" If F is significant, at least one predictor matters. Use summary(lm(...)) in R to see the F-statistic automatically.
Section 06 — The Crown Jewel

ANOVA — Analysis of Variance

F
Parametric · 3+ Groups · The Most Used Test in Plant Breeding
Analysis of Variance (ANOVA)

Why ANOVA Exists — The Multiple Testing Problem

Suppose you have 4 rice varieties. You might think: "I'll just run t-tests between all pairs!" That gives you 6 t-tests (4C2 = 6). Each test has a 5% chance of a false positive.

With 6 tests, the chance of getting at least one false positive = 1 - (0.95)⁶ = 26.5%. That's unacceptable science.

ANOVA solves this with a single, unified F-test across all groups simultaneously, maintaining your 5% error rate.

🧠 Memory Anchor

"ANOVA partitions the total variation in your data into between-groups variation (signal — what we're interested in) and within-groups variation (noise — random error). If signal ≫ noise, F is large → significant."

The Fundamental Partition of Variance

Total Variation = Explained + Unexplained
SS_Total = SS_Between + SS_Within
SS_Total = Σ(yᵢⱼ - ȳ..)²
SS_Between (Treatment) = n × Σ(ȳⱼ - ȳ..)²
SS_Within (Error) = Σ Σ(yᵢⱼ - ȳⱼ)²

F = MS_Between / MS_Within = (SS_Between/df_between) / (SS_Within/df_within)

Assumptions of ANOVA

🔔
Within-group data are normally distributed (residuals are normal)
⚖️
Homogeneity of variances — all groups have similar variance
🔗
Observations are independent
📏
Response variable is continuous

R Code — One-Way ANOVA

Scenario: You are testing 5 maize varieties in a completely randomised design (CRD). 6 replications each. Do they significantly differ in grain yield?

R Language
oneway_anova.R — CRD with 5 varieties
# ════════════════════════════════════════════════════
# ONE-WAY ANOVA — COMPLETELY RANDOMISED DESIGN (CRD)
# 5 Maize varieties × 6 replications
# Response: Grain yield (t/ha)
# H₀: μ₁ = μ₂ = μ₃ = μ₄ = μ₅ (all means are equal)
# ════════════════════════════════════════════════════

# STEP 1: Build the dataset (long format — always use this!)
yield <- c(
  6.2, 6.5, 6.1, 6.3, 6.4, 6.2,  # Variety 1 (DKC-9108)
  7.1, 7.4, 7.0, 7.3, 7.5, 7.2,  # Variety 2 (P-3396)
  5.8, 5.5, 5.9, 5.7, 5.6, 5.8,  # Variety 3 (NK-6240)
  8.1, 8.3, 8.0, 8.2, 8.4, 8.1,  # Variety 4 (PAC-999)
  7.6, 7.8, 7.5, 7.7, 7.9, 7.6   # Variety 5 (Hishell)
)

variety <- factor(rep(
  c("DKC-9108", "P-3396", "NK-6240", "PAC-999", "Hishell"),
  each = 6
))

maize_data <- data.frame(variety, yield)
str(maize_data)  # always check your data structure!

# STEP 2: Descriptive statistics per variety
tapply(yield, variety, function(x)
  c(mean=round(mean(x),2), sd=round(sd(x),2), n=length(x)))

# STEP 3: CHECK ASSUMPTIONS before ANOVA
# 3a. Normality of residuals (Shapiro-Wilk)
model <- aov(yield ~ variety, data = maize_data)
shapiro.test(residuals(model))  # p > 0.05 = normal

# 3b. Homogeneity of variances (Bartlett's test)
bartlett.test(yield ~ variety, data = maize_data)
# p > 0.05 = equal variances ✓

# 3c. Diagnostic plots (4-panel check)
par(mfrow = c(2, 2))
plot(model)  # 4 plots: residuals vs fitted, QQ, scale-location, leverage
par(mfrow = c(1, 1))  # reset

# STEP 4: RUN ONE-WAY ANOVA
anova_result <- summary(model)
print(anova_result)

# STEP 5: POST-HOC TEST — Which pairs differ?
# ANOVA only tells you "at least one pair differs"
# Tukey's HSD tells you WHICH pairs differ
tukey_result <- TukeyHSD(model)
print(tukey_result)
plot(tukey_result, las = 1, col = "#3fb950")

# STEP 6: Compact Letter Display (CLD) — used in papers!
# install.packages("multcompView"); install.packages("emmeans")
library(multcompView)
library(emmeans)

emm  <- emmeans(model, ~ variety)
cld_table <- cld(emm, Letters = letters, reversed = TRUE)
print(cld_table)
# Varieties sharing the same letter are NOT significantly different
# e.g., "a" = best group, "b" = second group, "ab" = overlapping

# STEP 7: Publication-quality bar plot with error bars
library(ggplot2)
summary_df <- aggregate(yield ~ variety,
                          data = maize_data,
                          FUN  = function(x)
                            c(mean=mean(x), se=sd(x)/sqrt(length(x))))
summary_df <- do.call(data.frame, summary_df)
names(summary_df) <- c("variety", "mean", "se")

ggplot(summary_df,
       aes(x = reorder(variety, -mean), y = mean, fill = variety)) +
  geom_bar(stat = "identity", colour = "white", width = 0.65) +
  geom_errorbar(aes(ymin = mean - se, ymax = mean + se),
                width = 0.2, colour = "white") +
  labs(title  = "Maize Variety Trial — Mean Grain Yield",
       subtitle = "Error bars = ±SE | Letters denote Tukey's HSD groups",
       x = "Variety", y = "Grain Yield (t/ha)") +
  theme_minimal() +
  theme(legend.position = "none",
        plot.background = element_rect(fill = "#0d1117"),
        panel.background = element_rect(fill = "#0d1117"),
        text = element_text(colour = "white")) +
  scale_fill_manual(values = c(
    "#3fb950", "#58a6ff", "#d29922", "#bc8cff", "#f85149"))

R Code — RCBD (Randomised Complete Block Design)

Scenario: Same 5 varieties, but your field has a fertility gradient. You blocked perpendicular to the gradient. 4 blocks total. This is the most common design in plant breeding trials.

R Language
rcbd_anova.R — The bread & butter of field experiments
# ════════════════════════════════════════════════════
# RCBD (RANDOMISED COMPLETE BLOCK DESIGN) ANOVA
# 5 varieties × 4 blocks
# Blocking accounts for field heterogeneity
# Model: yield ~ variety + block
# ════════════════════════════════════════════════════

# STEP 1: Create dataset
rcbd_data <- data.frame(
  yield = c(
    6.2, 6.4, 6.1, 6.3,  # DKC-9108, Blocks 1-4
    7.1, 7.4, 7.0, 7.3,  # P-3396
    5.8, 5.5, 5.9, 5.7,  # NK-6240
    8.1, 8.3, 8.0, 8.2,  # PAC-999
    7.6, 7.8, 7.5, 7.7   # Hishell
  ),
  variety = factor(rep(
    c("DKC-9108","P-3396","NK-6240","PAC-999","Hishell"),
    each = 4)),
  block = factor(rep(1:4, times = 5))
)

# STEP 2: Fit RCBD model (block removes field heterogeneity)
rcbd_model <- aov(yield ~ variety + block,
                    data = rcbd_data)

# STEP 3: ANOVA table
summary(rcbd_model)
# Now you see THREE rows: variety, block, Residuals
# Block SS tells you how much field heterogeneity you removed!

# STEP 4: Efficiency of blocking
# Compare with CRD error MS to see if blocking helped
crd_model  <- aov(yield ~ variety, data = rcbd_data)
rcbd_anova <- summary(rcbd_model)
crd_anova  <- summary(crd_model)

cat("Error MS in CRD:",
    crd_anova[[1]]["Residuals", "Mean Sq"], "\n")
cat("Error MS in RCBD:",
    rcbd_anova[[1]]["Residuals", "Mean Sq"], "\n")
# Smaller error MS in RCBD = blocking was effective!

# STEP 5: Post-hoc — Tukey's HSD
TukeyHSD(rcbd_model, which = "variety")

# STEP 6: Check CV% of the experiment
grand_mean <- mean(rcbd_data$yield)
error_ms   <- rcbd_anova[[1]]["Residuals", "Mean Sq"]
CV_pct     <- (sqrt(error_ms) / grand_mean) * 100
cat("\nExperimental CV% =", round(CV_pct, 2), "%\n")
# CV < 10% for field trials = good experimental precision
# CV < 20% is acceptable in plant breeding

R Code — Two-Way ANOVA (Variety × Environment Interaction)

The most powerful and commonly published analysis. Do varieties respond differently across environments? This is G×E interaction — the core challenge of plant breeding.

R Language
twoway_anova_GxE.R — Genotype × Environment interaction
# ════════════════════════════════════════════════════
# TWO-WAY ANOVA: GENOTYPE × ENVIRONMENT (G×E)
# 3 Varieties × 2 Environments × 3 reps
# H₀₁: No variety effect
# H₀₂: No environment effect
# H₀₃: No G×E interaction (the crucial one in breeding!)
# ════════════════════════════════════════════════════

gxe_data <- data.frame(
  yield = c(
    # Env 1 (Irrigated): V1, V2, V3 (3 reps each)
    6.2, 6.5, 6.3,  7.1, 7.3, 7.0,  5.8, 5.6, 5.9,
    # Env 2 (Rainfed): V1, V2, V3 (3 reps each)
    5.1, 5.3, 5.0,  6.8, 7.0, 6.9,  4.2, 4.0, 4.3
  ),
  variety = factor(rep(rep(c("V1","V2","V3"), each=3), 2)),
  environment = factor(rep(c("Irrigated","Rainfed"), each=9))
)

# STEP 1: Full two-way ANOVA with interaction term
twoway_model <- aov(
  yield ~ variety * environment,  # * includes interaction
  data = gxe_data
)
summary(twoway_model)

# STEP 2: Interpret the interaction term first!
# If variety:environment is significant (p < 0.05),
# then G×E interaction exists — varieties respond
# DIFFERENTLY across environments
# This means you CANNOT recommend one variety for all
# environments — location-specific recommendations needed

# STEP 3: Interaction plot — the visual diagnostic
interaction.plot(
  x.factor    = gxe_data$environment,
  trace.factor = gxe_data$variety,
  response    = gxe_data$yield,
  fun         = mean,
  type        = "b",
  pch         = c(19, 17, 15),
  col         = c("#3fb950", "#58a6ff", "#d29922"),
  legend      = TRUE,
  xlab        = "Environment",
  ylab        = "Mean Yield (t/ha)",
  main        = "G×E Interaction Plot",
  lwd         = 2
)
# Parallel lines = NO interaction (consistent across envs)
# Crossing lines  = INTERACTION (varieties swap rankings!)

# STEP 4: Simple effects — compare varieties within each env
library(emmeans)
emm2 <- emmeans(twoway_model, ~ variety | environment)
pairs(emm2)  # pairwise comparisons within each environment

# STEP 5: ggplot2 — professional interaction plot
library(ggplot2)
library(dplyr)

gxe_summary <- gxe_data %>%
  group_by(variety, environment) %>%
  summarise(
    mean_yield = mean(yield),
    se_yield   = sd(yield) / sqrt(n()),
    .groups = "drop"
  )

ggplot(gxe_summary,
       aes(x = environment, y = mean_yield,
           colour = variety, group = variety)) +
  geom_line(linewidth = 1.2) +
  geom_point(size = 4) +
  geom_errorbar(aes(ymin = mean_yield - se_yield,
                     ymax = mean_yield + se_yield),
                width = 0.1) +
  labs(title    = "G×E Interaction: Variety Performance Across Environments",
       subtitle  = "Crossing lines indicate significant G×E interaction",
       x = "Environment", y = "Mean Yield (t/ha)",
       colour = "Variety") +
  scale_colour_manual(
    values = c("#3fb950", "#58a6ff", "#d29922")) +
  theme_minimal(base_size = 14)
⚠️
Critical: ANOVA Is Significant — Now What? A significant ANOVA F-test only tells you "at least two groups differ." It does NOT tell you which groups differ. You MUST follow up with a post-hoc test:
• Tukey's HSD — best for comparing all pairs, controls familywise error
• Dunnett's test — comparing all treatments to a control
• LSD (Fisher's) — liberal, often used in plant breeding but inflates error
• Bonferroni — very conservative, good when few pre-planned comparisons

Non-Parametric Alternative — Kruskal-Wallis Test

When ANOVA assumptions are badly violated (especially non-normal data, ordinal scores, disease ratings), use the Kruskal-Wallis test — the non-parametric equivalent of one-way ANOVA.

R Language
kruskal_wallis.R
# ════════════════════════════════════════════════════
# KRUSKAL-WALLIS TEST
# Non-parametric alternative to one-way ANOVA
# Example: Disease severity scores (0-9 scale)
#          are ORDINAL — use Kruskal-Wallis
# ════════════════════════════════════════════════════

disease_score <- c(3,2,4,3,2,  # Variety 1
                    7,8,6,7,8,  # Variety 2
                    1,2,1,2,1)  # Variety 3
disease_var   <- factor(rep(c("V1","V2","V3"), each=5))

kw_result <- kruskal.test(disease_score ~ disease_var)
print(kw_result)

# Post-hoc for Kruskal-Wallis: Dunn's test
# install.packages("dunn.test")
library(dunn.test)
dunn.test(disease_score, disease_var,
          method = "bonferroni")  # or "bh" for BH correction
Section 07 — Your Lifetime Reference

The Ultimate Decision Guide

Bookmark this. Tattoo it. When in doubt, work through these questions in order.

🌿 Ask These Questions in Order

1
What type of data do I have?

Categorical/Counts → go to χ² test. Continuous → continue to step 2.

2
How many groups am I comparing?

1 group vs a standard → one-sample t or z. 2 groups → t or z. 3+ groups → ANOVA.

3
What is my sample size, and do I know σ?

n < 30 or σ unknown → t-test. n ≥ 30 AND σ known → z-test.

4
Are my two groups independent or paired?

Independent groups → independent t-test. Same subjects measured twice → paired t-test.

5
Am I comparing means or variances?

Means → t, z, ANOVA. Variances between two groups → F-test (var.test in R).

6
Are ANOVA assumptions met?

Normality + homogeneity → ANOVA. Violated assumptions → Kruskal-Wallis (non-parametric).

R Code — Complete Analysis Workflow (Copy & Adapt)

This is the recommended workflow for any new dataset. Run this sequence every time before choosing your test.

R Language
master_workflow.R — Run this before any statistical test
# ════════════════════════════════════════════════════
# MASTER ANALYSIS WORKFLOW FOR PLANT BREEDING DATA
# Copy this template for every new experiment
# ════════════════════════════════════════════════════

# ── 1. LOAD YOUR DATA ──────────────────────────────
# From CSV file:
data <- read.csv("your_experiment.csv", header = TRUE)
# Or use built-in for practice:
data <- data.frame(
  yield   = c(...),  # your numeric response
  variety = factor(c(...)),  # your factor groups
  block   = factor(c(...))   # your blocks if any
)

# ── 2. EXPLORE DATA STRUCTURE ───────────────────────
str(data)           # variables and types
head(data)          # first 6 rows
summary(data)       # descriptive stats instantly
dim(data)           # rows × columns
table(data$variety) # replication balance

# ── 3. VISUALISE BEFORE TESTING ─────────────────────
boxplot(data$yield ~ data$variety,
        col = rainbow(length(unique(data$variety))),
        main = "Data Distribution by Group",
        ylab = "Response Variable")

# ── 4. CHECK ASSUMPTIONS ────────────────────────────
# 4a. Normality per group
by(data$yield, data$variety,
   function(x) shapiro.test(x))

# 4b. Equal variances
bartlett.test(yield ~ variety, data = data)

# ── 5. CHOOSE AND RUN YOUR TEST ─────────────────────
# Based on your decision from the guide above:

# TWO GROUPS → t-test:
# t.test(yield ~ variety, data=data, var.equal=TRUE)

# 3+ GROUPS → ANOVA:
# model <- aov(yield ~ variety, data=data)
# summary(model)

# COUNTS/CATEGORIES → chi-square:
# chisq.test(observed, p=expected_probs)

# VARIANCES → F-test:
# var.test(group1, group2)

# ── 6. POST-HOC (for ANOVA) ─────────────────────────
# TukeyHSD(model)

# ── 7. REPORT RESULTS ───────────────────────────────
# Always report: F/t/χ² statistic, df, p-value, effect size
# Example: "Varieties differed significantly in grain yield
# (F₄,₂₅ = 390.29, p < 0.001, η² = 0.98)"

# ── η² (eta-squared) — effect size for ANOVA ──────
model  <- aov(yield ~ variety, data = data)
aov_ss <- summary(model)[[1]]
eta2   <- aov_ss["variety", "Sum Sq"] /
            sum(aov_ss[, "Sum Sq"])
cat("η² (eta-squared) =", round(eta2, 3), "\n")
# η² > 0.14 = large effect in biological sciences

How to Write Statistical Results in Your Research Paper

📝
Correct Ways to Report Results

t-test: "The grain yield of Variety A (x̄ = 46.7 g/plant) was significantly higher than Variety B (x̄ = 42.8 g/plant; t₂₂ = 8.14, p < 0.001, Cohen's d = 3.40)."

χ² test: "The observed F₂ segregation ratio (315:85) did not deviate significantly from the expected 3:1 Mendelian ratio (χ²₁ = 0.267, p = 0.605)."

ANOVA: "Varieties differed significantly in grain yield (F₄,₂₅ = 390.29, p < 0.001, η² = 0.98). Post-hoc Tukey's HSD analysis revealed that PAC-999 (8.18 t/ha) and Hishell (7.70 t/ha) outyielded all other varieties (p < 0.05)."

F-test: "The yield variance of DS228 (s² = 13.82) was significantly greater than JS335 (s² = 0.28), indicating lower stability (F₁₉,₁₉ = 0.020, p < 0.001)."
